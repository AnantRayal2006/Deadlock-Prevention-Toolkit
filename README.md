🚀 Deadlock Prevention Toolkit

An interactive Operating System project that visualizes and simulates Deadlock Prevention using Banker’s Algorithm. This toolkit helps users understand safe and unsafe states, detect deadlocks, and explore recovery strategies with step-by-step execution.

🎥 Demo Video

👉 https://youtu.be/PutXPGOaC-8


📌 Features
✅ Banker’s Algorithm Implementation
🎯 Safe / Unsafe State Detection
🔄 Step-by-Step Simulation (Animated)
⚠️ Deadlock Detection
♻️ Recovery Suggestion (Victim Selection)
📊 Resource Allocation Visualization (Chart.js)
📘 Educational Module (Coffman Conditions)
📤 Export Results as Report
🧠 Concepts Covered
Deadlock in Operating Systems
Coffman Conditions:
Mutual Exclusion
Hold and Wait
No Preemption
Circular Wait
Banker’s Algorithm
Deadlock Avoidance vs Prevention vs Detection
🛠️ Tech Stack
Frontend: HTML5, CSS3
Logic: JavaScript (ES2020)
Visualization: Chart.js
Tools: VS Code, GitHub
⚙️ How It Works
Enter:
Process names
Available resources
Allocation matrix
Max need matrix
Click:
▶️ Run Check → To check safe state
⏳ Step-by-Step → To visualize execution
♻️ Recover → To simulate recovery
Output:
Safe sequence OR
Deadlock detection with recovery suggestion
📊 Algorithm Used

Banker’s Algorithm

Computes Need = Max - Allocation
Checks if processes can execute safely
Generates safe sequence OR detects deadlock
